Javascript
Diseño y desarrollo web.-Javascript es un lenguaje de programacion que se utiliza con frecuencia en diseño web
para generar efectos mas complejos que no se puedan alcanzar usando HTML.Javascript se utiliza para una gran
variedad de usos, pero una de las aplicaciones mas comunes esta en la creacion de los botones de la navegacion que
cambian de color o aspecto cuando el usuario mueve el cursor sobre el botón.
HTML
Siglas de las palabras inglesas: Hypertext Markup languaje. Es decir, lenduaje marcado de hipertexto. Lenguaje
informatico para crear paginas web. Conjuntos de etiquetas o instrucciones que permiten estructurar el contenido
de una web e incluir los hipervinculos o enlaces a otras paginas.Este lenguaje lo invento en 1991 el Doctor
Berners-Lee del CERN en Suiza
